# modules/ai/

Lógica de negocio y utilidades relacionadas con IA:
- gemini-text.service.ts: generación de texto con Gemini
- gemini-vision.service.ts: análisis de imágenes con Gemini
- hf-image.service.ts: generación de imágenes con Hugging Face

Aquí se centralizan los servicios de IA para texto, visión e imágenes.
