# tests/

Tests unitarios y de rendimiento del proyecto. Ejecuta individualmente con `npx tsx <test-file>`.

### Lista de tests
- `admin-actions.test.ts` — Acciones del panel de administración
- `analytics.test.ts` — Servicio de analytics
- `contact-route.test.ts` — Endpoint de contacto
- `json-sanitizer.test.ts` — Sanitización de JSON de IA
- `utils.test.ts` — Utilidades generales
- `verify_fix.test.ts` — Verificación de correcciones
- `news-sources.bench.ts` — Benchmark de fuentes RSS (`npm run test:bench`)
